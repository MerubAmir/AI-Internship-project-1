📊 Iris Dataset Exploratory Data Analysis (EDA)
📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the classic Iris dataset to understand data distribution, feature relationships, and basic statistical patterns.

The goal is to practice:

Data understanding
Visualization techniques
Statistical insights
Python data analysis workflow

📂 Dataset Information

The dataset used is the Iris Dataset, which contains measurements of iris flowers.

Features:
Sepal Length
Sepal Width
Petal Length
Petal Width
Species (Target variable)

Classes:
Setosa
Versicolor
Virginica

Source: Common ML benchmark dataset used in classification problems.

🎯 Objectives
Understand dataset structure
Identify patterns between features
Visualize distributions and relationships
Detect potential outliers
Gain insights for classification problems

🛠️ Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
📊 Key Steps Performed
1. Data Loading
Imported dataset using Pandas
Checked structure and summary
2. Data Exploration
.head(), .info(), .describe()
Checked missing values
3. Data Visualization
Histogram (feature distribution)
Boxplot (outlier detection)
Scatter plot (feature relationships)
Pairplot (class separation visualization)
4. Insights Extraction
Relationship between petal length and width
Class separability based on features
Distribution patterns of each feature
📈 Key Findings
Petal features are more useful for classification than sepal features
Setosa is clearly separable from other classes
Some overlap exists between Versicolor and Virginica
No major missing data issues found
